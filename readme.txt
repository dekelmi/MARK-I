Документация MARK I



Описание: 

"MARK I" - интерактивный хелпер, разработанный на Node.js
Он предоставляет различные ответы на вопросы пользователя, используя различные модули, такие как работа с базой данных, работа с API и другие.



Запуск:

Для запуска хелпера запустите скрипт в терминале

'node mark1.js'



Поддерживаемые команды:

Введите 'help' чтобы открыть список доступных вопросов.



Примеры использования 

1. Задайте вопрос и получите ответ:

Задайте вопрос, и я постараюсь ответить на него:
> Как дела?
> Всё замечательно, как дела у Вас?


2.Получите информацию о расположении кода:

Задайте вопрос, и я постараюсь ответить на него:
> Где расположен твой код?
> Ссылка на GitHub репозиторий, где располагается код: "https://github.com/Kelmicode/MARK I"


3. Получите прогноз погоды:

Задайте вопрос, и я постараюсь ответить на него:
> Погода в городе Москва?
> (полученный актуальный статус погоды через сервис "openweathermap.org")



Примечания:

1. При вводе неизвестного вопроса, помощник сообщит:
"Такой вопрос не поддерживается:("

2. При задании вопроса о базе данных или погоде, ответ может занять некоторое время, т.к происходит обращение к соответствующим модулям и API
